## Usage

Run:

```sh
npm i -DE @ivangabriele/prettier-config prettier@^3.0.0
```

or:

```sh
yarn add -DE @ivangabriele/prettier-config prettier@^3.0.0
```

Then add this line to your `package.json`:

```json
{
  // ...
  "prettier": "@ivangabriele/prettier-config"
}
```
